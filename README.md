# Origami Simulator

An interactive tool for simulating how flat surfaces fold along creases. This project allows users to:

- Import and visualize origami crease patterns
- Simulate folding in real-time
- Apply custom textures to the folded surface
- Export models in various formats (FOLD, STL, OBJ)
- Visualize mountain and valley folds

## Features

- Real-time 3D visualization
- Custom texture mapping support
- Multiple simulation modes (dynamic, static, rigid)
- Interactive folding controls
- Edge and crease visualization
- Export capabilities

## Setup

1. Clone this repository
2. Open index.html in a web browser
3. Start experimenting with different origami patterns!
